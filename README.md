# J_R_R_Tolkien_Drevo_in_list
J R R Tolkien Drevo in list Book cover reproduction
![Cover](/Interface%202%20clone.jpg)

J R R Tolkien Drevo in list was released by Mladinska Knjiga as part of the Odisej series, since the cover is a slieve that gets often lost I made an approximate reproduction, to make my copy look nice.
I've printed it as a poster (using online services) the width of the cover must be set to 21cm, i've used 30 cm x40 cm poster printing services.
